# CollarAI - Pitch

## Problem (30 sec)

- Most meeting communication is non-verbal, but we cannot observe ourselves live.
- Poor posture and low engagement reduce professional presence.

## Solution (30 sec)

- Chrome extension that gives real-time body-language coaching.
- Uses vision AI to analyze posture, facial engagement, gestures, and appearance.

## Demo (2 min)

- Configure API key in popup
- Join Google Meet
- Show real-time alert
- End meeting and open summary report

## Technical (1 min)

- Manifest V3 architecture
- Content script frame capture + service worker analysis
- Claude/OpenAI provider abstraction
- Local storage sessions + Chart.js summary visualization

## Impact (30 sec)

- Helps remote professionals improve meeting presence.
- Strong foundation for coaching, team analytics, and multi-platform support.
