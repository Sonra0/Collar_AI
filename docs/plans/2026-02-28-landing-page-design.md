# Collar AI Landing Page Design

## Overview
Single-page static site for DSOC 2026 hackathon submission. Single HTML + CSS + JS, deployed to GitHub Pages.

## Sections
1. **Hero** — Title, tagline, CTA, demo video placeholder (YouTube/Loom embed slot)
2. **Problem → Solution** — 2-column: why this exists → what it does
3. **Features** — 4 glass cards: Real-time Analysis, Live Coaching, Post-Meeting Summary, Privacy-First
4. **How It Works** — 3-step animated flow: Install → Join Meet → Get Coaching
5. **Installation Guide** — Step-by-step with placeholder mockups (download zip, dev mode, load unpacked, configure API key)
6. **Tech Stack** — Chrome MV3, Claude/OpenAI Vision, Chart.js
7. **Footer** — GitHub link, DSOC 2026 badge, author credit

## Visual Style
- Dark base (#0a0a0f), neon cyan (#00f0ff), violet (#8b5cf6)
- Sora body, Fraunces headings (matches extension)
- Glass-morphism cards, floating particles in hero
- Scroll-triggered fade-in animations (IntersectionObserver)
- Smooth scroll navigation

## Hosting
GitHub Pages — no build step, push and serve.
